Порядок відступів (Правило годинника)

Запам'ятай порядок: Верх → Право → Низ → Ліво (як годинникова стрілка).

margin: 25px 50px 25px 50px; /* Це означає: */
Top: 25px
Right: 50px
Bottom: 25px
Left: 50px

Margin vs Padding

Поглянь на цей блок: Padding розширює колір фону (належить елементу), а Margin створює порожнечу навколо (відштовхує сусідів).

Контент
.box-model-demo { background-color: #6c5ce7; border: 4px solid #2d3436; /* Відстань ВІД тексту ДО рамки */ padding: 30px; /* Відстань ВІД рамки ДО сусідів */ margin: 40px; }
